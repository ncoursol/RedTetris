# RedTetris Setup

npx @vue/cli create front

```
pnpm init
pnpm install express
pnpm install socket.io
pnpm install socket.io-client
pnpm install cors
```

Front:
```
pnpm install
pnpm install vuex
pnpm install vue-router
```

# Run App

Back:
```
pnpm back
```
Front:
```
pnpm front
```

Build:
```
pnpm build
```
